10 Most asked Java Questions

  1. Difference between JRE, JDK, JVM?
    JRE stands for Java Runtime Environment: usually downloaded as Java software.
    The JRE consists of the JVM and supporting libraries.
    The JRE is the runtime component of Java software and is all we need to run any Java application.
    JDK stands for Java Development Kit: Is JRE + development tools (java compiler, debugger)
    JVM stands for Java Virtual Machine: It translates and executes the Java bytecode, It’s the entity which transforms Java to become a “portable language”


  2. What is the main method? Why do we need one in java? Do we have to have a main method in java?
    Main method is the starting point of an application
    Whenever we execute a program, the main() is the first function to be executed.
    It is not mandatory to have a main method in java, without main() our Java code will compile but won’t run.


  3. What is polymorphism? Types of Polymorphism?
    Ability of an object to take on many forms, allows us to perform a task in multiple ways.
    Combination of overloading and overriding is known as Polymorphism
    Types:
    Compile time polymorphism(Static binding) → Method overloading
    Runtime polymorphism (Dynamic binding) → Method overriding


  4. What is encapsulation?
    Protective shield that hides the data.
    This data will be hidden from the other classes, achieved by declaring all the variables in the class as private, can be accessed with getters and setters


  5. What is a garbage collector and what is a Garbage Collector?
    Garbage collection is the process of looking at memory and identifying which objects are in use and which are not and deleting unused objects.
    Once an object is created it uses some memory and the memory remains allocated until there are references for the use of the object.
    We use System.gc() to call the garbage collector.


  6. What is Java regular expression(Regex)?
    Is an API for defining String patterns that can be used for searching, manipulating and editing a text, provided under the java.util.regex package.
  7. What is the difference between pass-by-value and pass-by-reference?
    Pass by value is a mechanism of copying the function parameter value to another variable.
    Pass by reference is a mechanism of passing the actual parameters to the function.


  8. What is autoboxing and unboxing?
    Autoboxing is the automatic conversion that the Java compiler makes between the primitive types and their corresponding object wrapper classes.
    For example, converting an int to an Integer, a double to a Double
    Unboxing is converting an object of a wrapper type(Integer) to its corresponding primitive(int) value

  11. How many catch blocks can we have? Which catch block will get executed if you get an ArithmeticException?
    There can be many catch blocks for a single try block and it is not necessary that each try block must be followed by a catch block.
    It should be followed by either a catch block or a finally block.
    However only the catch block encountered first on the call stack that satisfies the condition for the exception will be executed for that particular exception, rest will be ignored.


  12. What is binary search and how is it working?
    Binary search will compare the target value starting from the middle of the list and it keeps continuing to compare every time from the middle element until finding the last two values.
    It is very fast comparing to linear search
